# Project-1-Interactive-Front-End-Web-Application
Project for an university course.


The goal of this project for me was to make a proof of concept website based on an addon to World of Warcraft I saw recently.

The purpose is to track aspirational gear in the game as a sort of wishlist.
I found the restrictions and goals of this project to make working on this very time consuming, so this will probably never be fully finished.
However, it functions as a proof of concept using a limited set of items, and should support local storage of the wishlist. 

# THIS IS NOT INTENDED TO BE USED AS AN ACTUAL WISHLIST.

If you are genuinely looking for a solution to this and happen to find this, the idea was based on this addon.
https://www.curseforge.com/wow/addons/dungeons-and-raids-loot-wishlist

## AI assistance:

I used Claude AI / ChatGPT experimentally to do a couple things:

- Acquire a couple of items from Wowhead for me to skip some manual labor.
- Guide me into a modernistic "gamer" type CSS and to consolidate it into a single file solution.

## How to run locally

1. Download the html file
2. Run in your desired environment/browser.

## What did I learn

- To make sure an API actually exists instead of assuming it does.
- I learned how to make arguably functional, simple databases
- Hopefully learned how to make client sided storage
- CSS is a pain.
- HTML, javascript and CSS combined make surprisingly good function.
- I may have put too much effort in, but I like the result.
